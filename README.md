# Who-Wants-to-Be-a-Millionaire
A PyGame implementation of "Who Wants to Be a Millionaire" with OpenAI GPT integration. Features multiple difficulty levels, lifelines (50:50, phone-a-friend, ask-the-audience), dynamic question generation, and a polished GUI. Built with OOP principles and modular design for maintainability. Includes sound effects, background music, and a clean UI.
